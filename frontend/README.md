# Blog Frontend Module

This is the frontend module for the blog application.

## Tech Stack
- React 18
- TypeScript
- Vite
- ESLint

## Getting Started

### Installation
```bash
npm install
```

### Development Server
```bash
npm run dev
```

### Build
```bash
npm run build
```

### Preview
```bash
npm run preview
```
